# PNGTuber Software

![PNGTuber Preview](pngtuber.gif)

A desktop PNGTuber app written in **C++17** using the **SFML** library

### Features
* **Real-time audio processing:** The character's mouth moves based on audio
* **Eye-blinking:** Blink blink
* **Mouse tracking:** The character hand moves based on how user had move (and click)
